# 🎧 AI Music App

An AI-powered music playlist generator with real-time playback.

## 🚀 Features
- AI-generated playlists using Groq
- Apple Music–style UI
- YouTube playback integration
- Dynamic music search

## 🛠️ Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Groq API (LLaMA 3.1)
- YouTube scraping API

## ⚡ How it works
1. User enters mood/prompt
2. AI generates playlist
3. App fetches YouTube video
4. Plays instantly in browser

npm install
npm run dev
